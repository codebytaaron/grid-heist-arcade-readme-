## Overview

Grid Heist Arcade is a browser-based strategy-action game played on a grid-based map. The player controls a thief who must collect loot and escape while avoiding guards with different AI behaviors.

The game emphasizes careful movement, tool usage, and decision-making rather than fast reflexes.

## Features

- Grid-based, turn-style movement  
- Multiple guard AI behaviors  
- Loot collection and score tracking  
- Tools such as smoke bombs, decoys, and keycards  
- Procedurally generated levels  
- Increasing difficulty over time  

## Tech Used

- JavaScript
- HTML Canvas
- Web Audio API
